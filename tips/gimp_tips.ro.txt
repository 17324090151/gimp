# - Text will appear in the dialog as it is in this file.  This is
#   done on purpose in order to have more freedom in the layout of the
#   tips than with automatic word-wrapping, but this also means that
#   you have to avoid excessively long lines in this file.
# - Tips should be concise: 3 lines or less.
# - Advice for translators to other languages: keep the original tips
#   as comments before the translated tips.  It will be easier for

# - Liniile precedate de '#' sunt comentarii.
# - Liniile goale sau cele de comentariu separă două sfaturi (ele nu
#   sunt ignorate).  Mai multe linii goale sunt considerate una singură.
#   Dacă doriţi în sfat o linie goală, puneţi un spaţiu sau un tab.
# - În dialog, textul va apărea ca în acest fişier.  La afişarea
#   sfaturilor acest lucru vă dă o libertate mai mare decât formatarea
#   automată a rândurilor, însă trebuie să evitaţi liniile excesiv de
#   lungi.
# - Sfaturile trebuie să fie concise: 3 linii sau mai puţin.
# - Sfat pentru traducători: păstraţi sfaturile originale drept

pe imagine. Şi nu vă îngrijoraţi, puteţi reveni după geşeli...

# Tips for beginners start here
# (for people who are not familiar yet with layers and image formats)
#
# Aici sunt sfaturi pentru începători
# (pentru cei care nu sunt familiarizaţi cu straturile şi formatele)
#
# You can get context-sensitive help for most of the GIMP's features by
# pressing the F1 key at any time.  This also works inside the menus.
#
În cele mai multe situaţii, apăsând F1 puteţi primi ajutor în context.
Aceasta funcţionează şi pentru meniuri.

# The GIMP uses layers to let you organize your image.  Think of them
# as a stack of slides or filters, such that looking through them you
# see a composite of their contents.
#
În GIMP, pentru a vă organiza imaginea puteţi folosi straturi.
Consideraţi-le ca pe un teanc de folii transparente sau filtre,
prin care dacă priviţi, vedeţi o imagine compusă.

# You can perform many layer operations by right-clicking on the text
# label of a layer in the "Layers, Channels and Paths" dialog
#
Puteţi face operaţii cu straturi dând un clic dreapta pe eticheta
text a stratului din dialogul "Straturi, canale şi trasee".

# When you save an image to work on it again later, try using XCF,
# the GIMP's native file format (use the file extension ".xcf").
# This preserves the layers and every aspect of your work-in-progress.
# Once a project is completed, you can save it as JPEG, PNG, GIF, ...

Când salvaţi o imagine pentru a relua lucrul ulterior, folosiţi
formatul nativ al GIMP, XCF (folosiţi fişiere cu extensia ".xcf").
Acesta reţine straturile şi toate informaţiile necesare lucrului.
Când lucrarea este gata, o puteţi salva ca JPEG, PNG, GIF, ...

# The layer named "Background" is special because it lacks transparency.
# This prevents you from adding a layer mask or moving the layer up in
# the stack.  You may add transparency to it by right-clicking in the
# "Layers, Channels and Paths" dialog and selecting "Add Alpha Channel".
#
Stratul numit "fundal" (BG) este special, iniţial el nu-i transparent.
Aceasta previne aplicarea unei măşti pe el sau deplasarea sa în sus
în teanc. Puteţi să-i adăugaţi transparenţă dând un clic dreapta în
dialogul "Straturi, canale şi trasee" şi selectând "Adaugă canal alfa".

# Most plug-ins work on the current layer of the current image.  In
# some cases, you will have to merge all layers (Layers->Flatten Image)
# if you want the plug-in to work on the whole image.
#
Multe pluginuri acţionează asupra stratului curent al imaginii. În
unele cazuri, pentru a face ca pluginul să acţioneze asupra întregii
imagini trebuie să combinaţi straturile (Straturi->Aplatizează imaginea).

# Not all effects can be applied to all kinds of images.  This is
# indicated by a grayed-out menu-entry.  You may need to change
# the image mode to RGB (Image->Mode->RGB), add an alpha-channel
# (Layers->Add Alpha Channel) or flatten it (Layers->Flatten Image).
#
Nu toate efectele pot fi aplicate tuturor tipurilor de imagini.
Este cazul imaginilor gri. Trebuie să schimbaţi tipul imaginii in RGB
(Imagine->Mod->RGB), să adăugaţi un canal alfa (Straturi->Adaugă canal
alfa) sau să-l aplatizaţi (Straturi->Aplatizează imaginea).

# Tips for intermediate users start here
# Aici sunt sfaturi pentru cei ce au progresat.
#
# You can drag and drop many things in the GIMP.  For example, dragging
# a color from the toolbox or from a color palette and dropping it into
# an image will fill the current image or selection with that color.
#
În GIMP puteţi să trageţi şi să plasaţi multe lucruri. De exemplu,
trăgând o culoare dintr-o trusă sau dintr-o paletă şi plasând-o într-o
imagine, imaginea sau selecţia se vor umple cu culoarea respectivă.

# When using a drawing tool (Paintbrush, Airbrush, or Pencil),
# Shift-click will draw a straight line from your last drawing
# point to your current cursor position.  If you also press Ctrl,
# the line will be constrained to 15 degree angles.
#
Cu uneltele de desenat (pensulă, aerograf sau creion), shift-clic
va trage o linie dreaptă din ultimul punct desenat până la poziţia
curentă a cursorului. Dacă apăsaţi simultan şi Ctrl, linia va fi
orientată la unghiuri multipli de 15 grade.

# The file selection dialog box has command-line completion with
# Tab, just like the shell.  Type part of a filename, hit tab, and voila!
# It's completed.
#
În caseta de dialog pentru selecţia fişierelor completarea liniei de
comandă se poate face cu TAB exact ca în shell. Introduceţi o parte
a numelui fişierului, daţi TAB şi iată, linia se completează!

# You can reassign shortcut keys on any menu by bringing up the menu,
# selecting a menu item, and pressing the new shortcut key combination.
# This is dynamic and is saved when you exit GIMP.
#
Puteţi redefini tastele de acces direct din orice meniu apelând meniul,
selectând un element de meniu şi apăsând noua combinaţie de taste.
Este dinamic, iar când terminaţi sesiunea GIMP combinaţia este salvată.

# You can use the middle mouse button to pan around
# the image, if it's larger than its display window.
#
Dacă imaginea este mai mare decât fereastra în care este afişată,
ea poate fi deplasată cu butonul din mijloc al mausului.

# Click and drag on a ruler to place a Guide on an image.  All
# dragged selections will snap to the guides.  You can remove
# guides by dragging them off the image with the Move tool.
#
Prin clic şi tragere pe riglă puteţi plasa pe imagine o linie
de ghidaj. Toate selecţiile trase se vor alinia la aceste linii.
Îndepărtarea liniilor se face prin mutarea lor în afara imaginii.

# You can drag a layer from the "Layers, Channels and Paths" dialog
# and drop it onto the toolbox.  This will create a new image
# containing only that layer.
#
Din dialogul "Straturi, canale şi trasee" puteţi trage un strat
şi să-l plasaţi în trusa de unelte. Aceasta va crea o imagine nouă,
conţinând doar acel strat.

# A Floating Selection must be anchored to a new layer or to the last
# active layer before doing other operations on the image.  Click on the
# New Layer or Anchor Layer buttons in the "Layers, Channels and Paths"
# dialog, or use the menus to do the same.
#
O selecţie flotantă trebuie ancorată la un strat nou sau la ultimul
strat activ înainte de a face o altă operaţie pe imagine. Pentru asta
faceţi clic pe butoanele Strat nou sau Ancorează strat din dialogul
"Straturi, canale şi trasee", sau folosiţi meniurile.

# The GIMP supports gzip compression on the fly.  Just add
# ".gz" (or ".bz2", if you have bzip2 installed) to the filename
# and your image will be saved compressed.  Of course loading
# compressed images works too.
#
GIMP suportă compresia gzip automat. Adăugaţi la numele
fişierului ".gz" (sau ".bz2" dacă aveţi bzip2 instalat),
iar imaginea va fi salvată gata comprimată. Fireşte,
încărcarea unei imagini comprimate funcţionează şi ea.

# Pressing and holding the Shift key before making a selection allows
# you to add to the current selection instead of replacing it.  Using
# Ctrl before making a selection subtracts from the current one.
#
Ţinerea apăsată a Shift înainte de a face o selecţie vă permite să
adăugaţi la selecţia curentă în loc de a o înlocui. Selectând cu Ctrl
apăsat eliminaţi selecţia din selecţia curentă.

# You can press or release the Shift and Ctrl keys while you are
# making a selection in order to constrain it to a perfect square
# or circle, or to have it centered on its starting point.
#
Când faceţi o selecţie puteţi apăsa sau elibera Shift sau Ctrl
până obţineţi de exemplu un pătrat sau un cerc perfect, sau pană
obţineţi centrarea la punctele de start.

# Using Edit->Stroke allows you to draw simple squares or circles by
# painting the edge of your current selection with the active brush.
# More complex shapes can be drawn with Filters->Render->Gfig.
#
Cu Editare->Tuşă puteţi trasa pătrate sau cercuri simple
prin pictarea cu pensula activă a marginii selecţiei curente.
Puteţi trasa forme mai complexe cu Filtre->Randare->Gfig.

# If you stroke a path (Edit->Stroke), the current drawing tool and its
# settings are used.  You can use the Paintbrush in gradient mode, the
# Clone tool with a pattern or even the Eraser or the Smudge tool.
#
Când trasaţi o tuşă (Editare->Tuşă), se folosesc unealta de desenare
curentă şi setările sale. Puteţi folosi pensula în mod gradient,
unealta de clonare cu modele, sau uneltele de ştergere şi pătare.

# You can create and edit complex selections using the Bezier tool.
# The "Paths" tab in the "Layers, Channels and Paths" dialog allows
# you to work on multiple paths and to convert them to selections.
#
Puteţi face şi modifica selecţii complexe folosind unealta Bezier.
Secţiunea "Trasee" din caseta de dialog "Straturi, canale şi trasee"
vă permite să lucraţi cu trasee multiple şi să le convertiţi la selecţii.

# You can use the paint tools to change the selection.  Click on the
# Quick Mask button at the bottom left of an image window.  Change your
# selection by painting in the image and click on the button again to
# convert it back to a normal selection.
#
Puteţi folosi unealta de pictare pentru a modifica selecţia. Faceţi clic
pe butonul de mască promptă din stânga-jos a imaginii. Modificaţi selecţia
pictând pe imagine, apoi apăsaţi din nou butonul pentru a face conversia
înapoi la o selecţie normală.

# You can save a selection to a channel (Select->Save to Channel) and
# then modify this channel with any paint tools.  Using the buttons in
# the "Channels" tab of the "Layers, Channels and Paths" dialog, you can
# toggle the visibility of this new channel or convert it to a selection.
#
Puteţi salva selecţia într-un canal (Selecţie->Slvează în canal), iar apoi
puteţi modifica acest canal cu orice unealtă de pictat. Folosind butoanele
din secţiunea "Canale" din caseta de dialog "Straturi, canale şi trasee"
puteţi comuta vizibilitatea canalului sau să-l convertiţi la o selecţie.

# Tips for advanced users start here
# (this is mostly for learning shortcut keys)
# Aici sunt sfaturi pentru avansaţi
# (în special pentru învăţarea acceleratoarelor)
#
# If your screen is too cluttered, you can press Tab multiple times
# in an image window to hide or show the toolbox and other dialogs.
#
Dacă ecranul este prea dezordonat, puteţi apăsa de mai multe ori Tab
în fereastra cu imaginea pentru a ascunde sau afişa trusa de unelte
sau alte casete de dialog.

# You can adjust the selection range for fuzzy select
# by clicking and dragging left and right.
#
Puteţi ajusta domeniul unei selecţii difuze
prin clic şi deplasare stânga şi dreapta.

# Shift-click on the eye icon in the Layers dialog to hide all
# layers but that one.  Shift-click again to show all layers.
#
Shift-clic pe iconul ochiului din caseta de dialog a straturilor
ascunde toate straturile cu excepţia celui curent.
Un nou Shift-clic va arăta toate straturile.

# Ctrl-click on the layer mask's preview in the Layers dialog 
# toggles the effect of the layer mask.
#
Ctrl-clic pe previzualizarea măştii stratului în caseta de dialog
a straturilor comută efectul măştii stratului.

# Alt-click on the layer mask's preview in the Layers dialog 
# toggles viewing the mask directly.
#
Alt-clic pe previzualizarea măştii stratului în caseta de dialog
a straturilor comută pe afişarea directă a măştii.

# You can use Alt-Tab to cycle through all layers in an image 
# (if your window manager doesn't trap those keys...).
#
Dacă managerul ferestrelor nu foloseşte combinaţia Alt-Tab,
o puteţi folosi pentru a parcurge ciclic straturile imaginii.

# Ctrl-click with the Bucket Fill tool to have it use 
# the background color instead of the foreground color.
#
Ctrl-clic pe unealta de umplere determină folosirea culorii
de fundal (BG) în loc de culoarea de prim plan (FG).

# Control-drag with the Transform tool in rotation mode 
# will constrain the rotation to 15 degree angles.
#
Ctrl-clic pe unealta de transformare lucrând în mod rotire
determină rotirea cu unghiuri fixe de 15 grade.

# You can adjust and re-place a selection by using Alt-drag.
#
Cu Alt-tragere puteţi ajusta şi plasa din nou o selecţie.

# If your fonts turn out blocky, that's because they're not scalable 
# fonts. Most X servers support scalable Type 1 Postscript fonts.  
# Download and install them.  Some font servers allow you to use 
# TrueType (.ttf) fonts, which are also scalable.
#
Dacă în loc de litere vedeţi pătrăţele este din cauză ca nu aveţi
fonturi scalabile. Multe servere X admit fonturi postscript de tip 1.
Descărcaţi-le şi instalaţi-le. Unele servere permit şi fonturi
true-type (.ttf), care de asemenea sunt scalabile.

# To create a perfect circle, hold Shift while doing an ellipse select. To
# place a circle precisely, drag horizontal and vertical guides tangent to
# the circle you want to select, place your cursor at the intersection
# of the guides, and the resulting selection will just touch the guides.
#
Pentru a desena un cerc perfect apăsaţi Shift când desenaţi o elipsă.
Pentru selecta o zonă circulară precisă plasaţi linii de ghidaj
orizontale şi verticale tangente la zonă, apoi pornind din intersecţia
liniilor cercul de selecţie va fi tangent la liniile de ghidaj.

# If some of your scanned photos do not look colorful enough, you can
# easily improve their tonal range with the "Auto" button in the Levels
# tool (Image->Colors->Levels).  If there are any color casts, you can
# correct them with the Curves tool (Image->Colors->Curves).
#
Dacă culorile fotografiilor scanate nu arată prea bine puteţi
îmbunătăţi gama de tonuri cu butonul "Auto" al uneltei de nivele
(Imagine->Culori->Nivele). Dacă există devieri de culoare
corectaţi prin ajustarea curbelor (Imagine->Culori->Curbe).

# (end of tips)
