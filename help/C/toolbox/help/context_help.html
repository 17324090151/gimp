Context Help

Pressing Shift-F1 will invoke the context sensitive help system of The GIMP. This system will be shown as active when the mouse pointer is changed to a ? pointer. Clicking items with this pointer will display the relevant help page if one exists.

Additional Information

Default Keyboard Shortcut: Shift-F1
