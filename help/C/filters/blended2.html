Blended

Overview

The Blended plugin renders a blended beveled logo from black text. The text must be placed on an alpha layer.

Blended Options

Blended Settings

Offset

The Offset setting specifies the number of pixels that will be used to offset the shadow for the logo.

Background Color

This option sets the color used for the background of the operation.

Blend Mode

The Blend Mode dropdown selects the gradient mode.

Start Blend

This color selection button sets the start color for the blend.

End Blend

This color selector button sets the end color for the blend.

Gradient

This option selects the gradient that will be used by the plug-in.
