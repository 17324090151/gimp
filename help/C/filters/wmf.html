WMF

Overview

The GIMP can read files in the WMF (Windows Meta File) format. This format is often used for stock images (clip-art) for office applications on the Microsoft Windows platform. Although WMF is a vector format, GIMP rasterizes (converts to pixels) the image when the file is loaded.
