Spread

Overview

The Spread filter moves pixels in a random direction by a user specified amount.

Spread Options

Spread Settings

Spread Amount

The only available setting for this filter is the amount of spread to apply to the pixels.

  • Horizontal / Vertical: This sets the amount of spread across the named axis. The axis can be locked by clicking the Chain icon.

  • The unit that defines the distance that pixels will be moved is set using the dropdown list. By default it is set to Px or pixels.
