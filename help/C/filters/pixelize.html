Pixelize

Overview

The Pixelize filter renders the image using large color blocks. It is very similar to the effect seen on television when obscuring a criminal during trial.

Pixelize Options

The single option available when using Pixelize is used to interactively change the resulting block size. The number chosen will represent the size both horizontally and vertically of the rendered blocks. The color for each block is determined at render time by color weighting the underlying pixels, and determining the most used color.
