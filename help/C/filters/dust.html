Add Dust

Overview

The Add Dust filter adds curled, grey dust-like elements to the active layer.

Dust Options

Dust Settings

Density

Density controls the amount of dust elemnts that will populate the layer after rendering. A setting that is too high here can become extremely processor intensive, so adjust this setting carefully.

Seed

Seed is the numeric randomization base. Changing this number will result in different dust element shapes and positions.

Length

Length controls the length of the dust artifacts. A higher setting here will result in longer curls of dust.
