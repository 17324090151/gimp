Tileable Blur

Overview

The Tileable Blur filter can be utilized to blur already tileable images so that the tileable properties of the layer are not lost after blurring.

Tileable Blur Options

Tileable Blur Settings

Radius

The Radius slider sets the radius of the blur in the same fashion as the Gaussian Blur IIR , or the Gaussian Blur RLE filters.

Blur Vertically

This option sets the blur to work on the vertical axis.

Blur Horizontal

This option sets the blur to work on the horizontal axis.

Blur Type

This dropdown box sets the blur type to either IIR or RLE.
