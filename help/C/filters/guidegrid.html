Guide Grid

Overview

The Guide Grid filter places a grid arrangement of guides on the image.

Guide Grid Options

Guide Grid Settings

X spacing

This option sets the number of pixels between vertical guides.

Y spacing

This option sets the number of pixels between horizontal guides.

X offset

This setting adjusts the starting point for the horizontal guides.

Y offset

This setting adjusts the starting point for the vertical guides.

Remove old guides

The Remove existing guides checkbox will remove all guides from the image before laying down the new grid.
