Noisify

Overview

This filter adds RGB noise to the image area.

Noisify Options

Noisify Settings

Independent

This checkbox will toggle the forcing of the three color sliders to be moved individually or together. If this option is set the rendered noise will be white.

Red / Green / Blue

The three color sliders will increase or decrease the amount of the specified color in the noise render. If the Independent checkbox is not checked, the noise will contain elements of each color individually.

See also

Further discussion of noise can be found in the glossary under the Noise entry.
