Colorify

Overview

The Colorify filter remaps all color pixels in the active selection or layer to a chosen color. It preserves the luminosity of the image.

Colorify Options

Colorify Settings

Color Wells

The filter provides seven predefined colors for convenient selection, but another color can be selected by clicking Custom Color. A color selection dialog will open in which the desired color can be selected. The color contained in the Custom Color well is the actual color used for the filter.
