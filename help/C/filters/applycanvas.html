Apply Canvas

Overview

This filter applies a canvas-like effect to the current image. It textures the image as if it were an artist's canvas.

This filter is comparable to the GIMPressionist filter .

The Apply Canvas Options

Apply Canvas Settings

Direction

The four selectable diretions make very little difference to the final render. The Direction sets the starting direction of the canvas render.

Depth

The Depth slider controls the apparent depth of the rendered canvas effect. A higher setting here will render the canvas in a more intense fashion.
