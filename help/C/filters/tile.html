Tile

Overview

"Tile" creates multiple copies of the current image in a tiled pattern. The size of the tiled image should be set as a multiple of the current image to get a cleanly tiled image. For example, setting 400 x 400 for a 100 x 100 image would make the new image contain 4 copies of the current image.

It is often useful to select Create New Image so that the current image remains unaltered.
