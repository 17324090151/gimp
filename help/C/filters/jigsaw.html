Jigsaw

Overview

The Jigsaw filter renders a simple jigsaw pattern to the current layer.

Jigsaw Options

The Number of Tiles settings control both the number of Horizontal tiles, and the number of Vertical tiles.

The Bevel Edges settings allow control over the rendering of the edges of the tiles. Bevel Width sets the width of the shading on each tile, and Highlight controls the lighting aggresiveness. A higher Highlight setting, will produce tiles with more light.

Jigsaw Style can be set to either the traditional Square tile type, or the more recent Curved tile.
