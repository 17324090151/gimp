TeX String

Overview

The TeX String filter renders a TeX string as a new floating layer. For more information on TeX, visit the Comprehensive TeX Archive Network.

TeX String Settings

Input file is where the name of the TeX macro file should be inserted. Tex string is where the actual TeX String should be input.

Dpi sets the rendering DPI level.

Magstep sets the TeX magstep.

Anti aliasing controls the level of antialiasing that will be applied to the string at rendertime.
