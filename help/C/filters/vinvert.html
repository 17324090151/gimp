Value Invert

Overview

The "Value Invert" plug-in inverts the Value (brightness) part of an HSV (Hue Saturation Value) image. The Hue and Saturation of the image remain unaffected so color will be the same, just a different shade.

See also

For straight out color inversion, see the Invert help page.
