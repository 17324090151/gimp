Smooth Palette

Overview

This plugin will create a new image that contains palette information derived from the original image. It will attempt to create a smooth palette. That is, one that contains a blend between colors.

Smooth Palette Options

Smooth Palette Settings

Parameter Settings

There are three available options to the user.

  • Width: Specifies the width in pixels of the derived palette.

  • Height: Specifies the height in pixels of the derived palette.

  • Search Time: Dictates how long the filter should spend matching colors to derive a better, smoother palette.
