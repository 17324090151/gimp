Fit Text

Overview

The Fit Text filter renders a text string to a selection area. This filter requires a selection.

Fit Text Options

Fit Text Settings

Font

This button will launch the font browser allowing the selection of the typeface or font that will be used by this filter.

String

Any text entered into this input box will be used in the final render for this filter.
