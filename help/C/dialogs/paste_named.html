Paste Named

This command pastes image data from a selected Named Buffer in one of three methods: Paste , Paste Into , or Paste As New .

The dialog displayed by the Paste Named command can be used to manage the current Named Buffers. Buffers can be deleted using the Delete button and the paste operation can be cancelled by clicking the Cancel button.

Clicking cancel after deleting a Buffer will not reverse the delete operation.

Additional Information

Default Keyboard Shortcut: Ctrl-Shift-V
