The Navigation Window

The Navigation Window is designed to offer easy movement around the active image if the zoom is set higher than what the image window can display. If this is the case, there is an inversely colored rectangle that shows the location of the current view area in repspect to the image. This rectangular outline can be dragged to change the viewing region. Below the image window is shown the current zoom ratio and a zoom slider that allows easy zoom level control. The two buttons to the left and right of the ratio provide single click zoom controls.

Additional Information

Default Keyboard Shortcut: Ctrl-Shift-N
