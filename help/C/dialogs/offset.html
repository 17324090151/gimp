Offset

The Offset command is used to move layers in a mathematical fashion.

Offest Options

Offset X/Y

These two input boxes allow specification of the distance to move the layer. The default unit is pixels.

Wrap Around

If this option is checked areas that move off the image boundary will be wrapped to the opposing side.

Fill Type

If Wrap Around is not being used, the area that remains when a layer is moved can be replaced with either the Background color fill or a Transparent fill.

Offset by (x/2),(y/2)

This button will automatically fill the X and Y Offset input boxes with values that will move the layer half of its width right and half of its height down. This is useful for working with seamless tiles.

Additional Information

Default Keyboard Shortcut: Ctrl-Shift-O
