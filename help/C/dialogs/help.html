Help Page for Help

The internal help browser for The GIMP has three tabs located at the top of the window. Index, Contents, and the current page that this text is displayed in.

The Index tab displays links to the entire help system in an indexed format. The list forms a hierarchy of links that ensure fast access to specific subjects.

The help system can be invoked by pressing F1 while one of The GIMP dialogs or windows is focussed, pressing F1 while a menu entry is active, or by using the context sensitive help system.

Additional Information

Default Keyboard Shortcut: F1
