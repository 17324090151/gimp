The Info Window

The General Tab

Dimension (w x h)

Shows the image height and width both in pixels and in the current unit.

Resolution

Displays the resolution of the active image.

Scale Ratio

Shows the zoom factor of the active image.

Display Type

Shows the image mode of the active image.

Visual Class

Shows the visual class of the active display.

Visual Depth

Shows the visual depth of the active display.

The Extended Tab

This tab shows color information not unlike the Color Picker Tool but without changing the foreground color. For information on interpretation of the values, see the glossary entry for RGBA .

Additional Information

Default Keyboard Shortcut: Ctrl-Shift-I
