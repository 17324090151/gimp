Copy

The Copy command allows the user to make a copy of the current selection, which is then stored in The GIMP Clipboard . The information can be recalled using the using the Paste , Paste Into , and Paste As New commands. If no selection is made, The GIMP will copy the entire current layer.

Copy will only work on the current active layer. Any layers above or below the active one will be ignored.

Additional Information

Default Keyboard Shortcut: Ctrl-C
