Redo

Acting as the reverse of the Undo function, Redo reverses the last undo that was performed on an image. Redo acts as an undo for undo.

As with the Undo function, the user may wish to investigate the usefulness of the Undo History dialog which performs undo and redo operations in an interactive fashion.

Additional Information

Default Keyboard Shortcut: Ctrl-R
