Paste

The Paste function puts whatever is in The GIMP Clipboard from the last Copy or Cut command into the current image. The pasted section will be placed as a Floating Selection and is shown as a separate layer in the Layers Dialog .

If there is an existing selection on the canvas, it will be used to align the pasted data. Any data pasted when a selection already exists will be pasted using the selection as a center point. If the selection is to be used as a clipping region for the pasted data, this can be achieved by using the Paste Into command.

You can only have one floating selection at any one time and cannot work on a layer while there is a floating selection.

Additional Information

Default Keyboard Shortcut: Ctrl-V
