Undo

The Undo function allows the user to revert an image back one step in the drawing or editing process. Any one function performed against the image may be undone by clicking Undo. Further undo operations may be performed according to the number of undo levels that are configured in the Preferences Dialog .

If prefered, the user may wish to examine the usefulness of the Undo History dialog which allows interactive travel through the undo chain.

Additional Information

Default Keyboard Shortcut: Ctrl-Z
