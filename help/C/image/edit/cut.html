Cut

The Cut function allows the user to delete a selection save it to The GIMP Clipboard . The information can be recalled using the Paste , Paste Into , or Paste As New commands. If no selection is made, The GIMP will cut the entire current layer.

Cut will only work on the current active layer. Any layers above or below the active one will be ignored.

Additional Information

Default Keyboard Shortcut: Ctrl-X
