Convert to Grayscale

Converting images to grayscale is not unlike the effect produced by using the Desaturate function. The primary difference is that converting an image to grayscale results in the image having the color channels removed. Thus, an image that has been converted to grayscale cannot have color added to it.

Additional Information

Default Keyboard Shortcut: Alt-G
