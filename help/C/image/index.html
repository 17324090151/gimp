Image Menu Introduction

This chapter delves into the Image menu and the submenus therein. Basic functionality explanations can be found here starting with file operations and moving through selections, navigation, guides, and color manipulations.

The information here contains valuable insight into The GIMP as this menu comprises the most used area of the application. Some time should be dedecated to this part of the help system as it will give the user a solid understanding of The GIMP.
